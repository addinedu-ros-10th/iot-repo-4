## 🔍 확인 방법

### 1. 웹 브라우저 직접 접근
각 URL을 브라우저에 입력하여 접근 가능 여부 확인

### 2. HTTP 상태 코드 확인
- **200**: Repository 존재
- **404**: Repository 존재하지 않음
- **403**: 접근 권한 없음
- **기타**: 오류 또는 리다이렉트

### 3. README 내용 확인
존재하는 Repository의 경우 README.md 파일 내용 확인
